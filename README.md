### 一、相关技术
- 后端：`Java`、`JavaWeb` / `Springboot`。
- 前端：`Vue`、`HTML` / `CSS` / `Javascript` 等。
- 数据库：`MySQL`

### 二、相关软件（列出的软件其一均可运行）
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- 等

### 三、功能描述
系统分为（身份）：普通用户、管理员。

**普通用户功能：**
1. 登录
2. 修改密码
3. 个人信息
4. 供应商信息管理
5. 商品管理
6. 收银管理
7. 收银列表管理


**管理员功能：**
1. 登录
2. 个人中心（修改密码、个人信息）
3. 基础数据管理（供应商类型管理、所在地区类型管理、供应商信用等级类型管理、商品类型管理）
4. 供应商信息管理
5. 商品管理
6. 采购管理
7. 收银管理、收银列表管理
8. 员工管理
